# Orchard

> A Vue.js project with vue 2.0, vue-router and vuex starter kit for server side rendering.

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```

## Hoodie Setup
``` mkdir -p .hoodie/data/hoodie-config ```
